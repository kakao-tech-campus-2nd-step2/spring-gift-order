[[overview_http_verbs]]
=== HTTP verbs


|===
| Verb | Usage

| `GET`
| 리소스를 검색하는 데 사용됩니다

| `POST`
| 새 리소스를 생성하는 데 사용됩니다

| `PUT`
| 기존 리소스를 업데이트하는 데 사용됩니다

| `DELETE`
| 기존 리소스를 삭제하는 데 사용됩니다
|===

[[overview_http_status_codes]]
=== HTTP status codes


|===
| Status code | Usage

| `200 OK`
| 요청이 성공적으로 처리되었습니다

| `201 Created`
| 새 리소스가 성공적으로 생성되었습니다

| `204 No Content`
| 요청이 성공적으로 처리되었지만 반환할 내용이 없습니다

| `400 Bad Request`
| 요청이 잘못되었습니다. 응답 본문에 자세한 오류 정보가 포함됩니다

| `403 Forbidden`
| 클라이언트가 해당 리소스에 대한 필요한 권한이 없습니다. 인증된 사용자이지만 접근 권한이 없는 경우 발생합니다

| `404 Not Found`
| 요청한 리소스를 찾을 수 없습니다

| `500 Internal Server Error`
| 서버에서 오류가 발생했습니다. 가능한 한 이 오류를 피하도록 노력합니다
|===
